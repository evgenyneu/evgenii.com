Robot commands

  • Use "PLACE x,y,direction" command to put the robot on the board, where x,y are coordinates and 'direction' the direction the robot is facing, which can be SOUTH, NORTH, EAST or WEST. Example: PLACE 1,1,NORTH.
  • MOVE commands moves the robot.
  • LEFT, RIGHT commands turn the robot.
  • REPORT command outputs the current position and direction the robot is facing.
